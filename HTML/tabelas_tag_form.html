Alguns atributos da tag <form>

action Define a ação a ser executada quando o formulário for enviado
target _self : Abre o documento na mesma janela/guia em que foi clicado.
_blank : Abre o documento em uma nova janela ou guia.
method GET : Os dados do formulário são enviados como variáveis de URL.
POST : Os dados do formulário são enviados como transação de postagem HTTP.
autocomplete on : Ativa o preenchimento automático.
off : Desativa o preenchimento automático.

Alguns elementos subordinados à tag <form>

Elemento <input> Descrição
type="checkbox"Define um botão de escolha, do tipo caixa, para seleção de VÁRIOS valores.
type="color"Define um campo de entrada para escolha de cor.
type="date"Define um campo de entrada do tipo data.
type="datetime-local"Define um campo de entrada do tipo data e hora.
type="email"Define um campo de entrada do tipo endereço de e-mail.
type="file"Define um campo de seleção de arquivo e um botão "Procurar".
type="hidden"Define um campo de entrada não visível ao usuário.
type="image"Define uma imagem como botão de envio.
type="month"Permite que o usuário selecione um mês e ano.
type="number"Campo de entrada do tipo número.
type="password"Campo de entrada de senha.
type="radio"Botão de escolha do tipo rádio (único valor).
type="range"Campo de entrada do tipo controle deslizante.
type="reset"Botão para limpar todos os campos do formulário.
type="submit"Botão para enviar dados do formulário.
type="tel"Campo de entrada para número de telefone.
type="text"Campo de entrada de texto de linha única.
type="time"Campo que permite ao usuário selecionar um horário.
type="url"Campo de entrada que deve conter um endereço de URL.
